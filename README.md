
# Enquete com node

Enquete para votos em tempo real, com funcionalidade de deleção de votos antigos.

## Stack Utilizada

* Typescript
* Node
* fastify
* prisma
* redis
* postgress
* websocket
* insomnia
* docker

## Instalação


```bash
  npm init
  npm install

  npm run dev
  docker-compose up -d

  npx prisma studio
```
